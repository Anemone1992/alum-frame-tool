# 铝型材框架下料助手

一个本地运行的铝型材框架下料和 BOM 辅助工具。它用外尺寸、贯通方向和附加模块来生成型材切割清单、五金估算、板材/配件备注，以及可导入 3D 软件验证的简化 OBJ/STL 模型。

它不是完整 CAD，也不是强度校核工具；目标是减少手算下料、少买/错买配件、尺寸扣料不一致这些真实下单前最容易发生的问题。

## 使用方式

直接打开 `index.html` 即可。如果浏览器不允许打开本地文件，可以在这个目录运行：

```powershell
python -m http.server 8765 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:8765/index.html
```

## 当前支持

- 两步工作流：
  - 先确认外框
  - 再添加结构扩展、内部框架、抽拉/开合、面板安装、加固支撑、底脚/固定等模块
- 输入外宽 X、外深 Y、外高 Z
- 选择 2020、3030、4040、4545 型材
- 选择主体贯通方向：
  - 立柱贯通：高度不扣料，宽/深方向扣两端型材宽度
  - 横梁贯通：宽度不扣料，深/高方向扣两端型材宽度
  - 深梁贯通：深度不扣料，宽/高方向扣两端型材宽度
- 添加主框横撑高度
- 添加宽向或深向分隔立柱
- 按三轴角节点、二轴边节点、横撑/模块支撑点分别估算连接件
- 添加/删除附加模块，模块会进入 JSON 和 CSV 备注
- 结构扩展、内部框架、门框类模块会生成简化方框型材，并合并到 BOM、预览、OBJ/STL
- 面板类模块进入板材/面板清单
- 抽拉/开合、加固、底脚等非框架模块进入配件/待确认清单
- 支持导入之前导出的 JSON 项目
- 附加模块会按类型显示合适的挂载位置、默认尺寸和是否可重复
- 顶部延伸/悬挑会扩展现有顶框，避免重复生成重叠框
- 导出：
  - BOM CSV
  - JSON
  - OBJ 简化模型
  - STL 简化模型

## 注意

OBJ/STL 只是用于空间验证的简化方管模型，不包含真实 T 槽截面、孔位、攻丝、角码和螺丝模型。

五金数量是粗估，用来提醒采购，不应替代最终装配检查。真实下单前仍要确认角码类型、T 型螺母规格、螺丝长度、端盖数量和是否需要攻丝。
